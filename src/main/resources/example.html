This page demonstrates storing a persistent value in your browser using evercookie-applet.

Make sure you have the Java plugin installed, and make sure you accept any permission requests.

IMPORTANT! This applet is generally useless if it is unsigned. Even if you sign it, users may still be prompted to run it. The only real way to run this applet quietly would be to sign it with a certificate that the JVM trusts already, or that the user has opted to trust.

This is quite possible, but we'll leave it as an exercise for you. ;)
